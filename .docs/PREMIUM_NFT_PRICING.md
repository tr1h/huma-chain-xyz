```

   100 NFT < 10,000 NFT
   → Наши должны быть ДОРОЖЕ!

2. Perceived Quality:
   $60 NFT > $2 NFT
   → "Если дорого = значит круто"

3. Investor Appeal:
   $60 → $300 = ×5 рост
   $2 → $24 = ×12 рост, но кого волнует $22 профита?
   
   Лучше:
   $60 → $300 = $240 профит! 💰

4. Floor Price:
   Start $60 → Floor может быть $100-200
   Start $2 → Floor может быть $5-10
   
   Что лучше продаётся на вторичке?

5. Community Quality:
   $60 barrier → серьёзные игроки
   $2 barrier → флипперы и скамеры
```
