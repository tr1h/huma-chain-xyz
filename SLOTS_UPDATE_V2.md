## 📈 ВЛИЯНИЕ НА ИГРОКОВ:

### Психология:
```
1. Видят алерты регулярно → "Люди выигрывают!"
2. Джекпот реально достижим → "Я тоже могу!"
3. Разные типы алертов → интересно следить

```

### Экономика:
```
Джекпот пул растет быстрее:
- Больше игроков → больше ставок
- Больше ставок → больше 5% в пул
- Больше пул → больше мотивация играть
- Цикл повторяется! 🔄
```
