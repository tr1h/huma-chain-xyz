## 🎯 Почему 5 тиров лучше чем 3?

### Психология рарности:

```
3 тира:
Bronze (90%) → "Обычное"
Silver (9%) → "Редкое"
Gold (1%) → "Очень редкое"

Проблема: Большой скачок между Silver и Gold
Silver (9%) vs Gold (1%) = ×9 разница!

5 тиров:
Common (90%) → "Основное"
Uncommon (7%) → "Не такое уж редкое"
Rare (2.6%) → "Редкое"
Epic (0.36%) → "Очень редкое"
Legendary (0.04%) → "ВАУ! ЛЕГЕНДА!"

Преимущества:
✅ Плавная прогрессия редкости
✅ Больше целей для игроков
✅ Больше хайпа ("У кого-то выпал Epic!")
✅ Больше торговли на вторичке
✅ Как в играх (Common→Legendary)
✅ Все понимают эту систему
```

### Маркетинг:

```
С 3 тирами:
"Минт Bronze NFT!"
→ Скучно

С 5 тирами:
"OMG! Кто-то заминтил LEGENDARY! 
Всего 2 во всей коллекции! 😱"
→ ВСЕ бегут минтить, надеясь на свой Legendary!

Твиттер взрывается:
"@user123 got LEGENDARY #1! 🔥🔥🔥
Only 1 more left in existence!"

→ FOMO ×1000!
```
