## 🎲 КАК РАБОТАЕТ ДЖЕКПОТ ПУЛ:

### 1. Наполнение пула:

```
Каждая ставка → 5% идет в джекпот пул

Пример:
- Игрок 1: ставка 1,000 → +50 в пул
- Игрок 2: ставка 2,000 → +100 в пул
- Игрок 3: ставка 500 → +25 в пул
Пул вырос: 0 → 175 TAMA
```

### 2. Выигрыш джекпота:

```
Кто-то выбил 🎰🎰🎰 → ЗАБИРАЕТ ВЕСЬ ПУЛ!

Игрок 4: ставка 2,000, выбил 🎰🎰🎰
Выигрыш:
- Базовый: 2,000 × 100 = 200,000 TAMA
- Джекпот пул: 175 TAMA
- ИТОГО: 200,175 TAMA

Пул сбрасывается: 175 → 0
```

### 3. Цикл повторяется:

```
Пул снова начинает наполняться с 0!
```

## 🎯 СТРАТЕГИЯ:

### Когда играть на джекпот?

```
1. Когда пул большой (50,000+)
   → Выгодно! Даже с маленькой ставкой можно много выиграть

2. Сразу после джекпота
   → НЕ выгодно, пул = 0

3. После 100+ игр без джекпота
   → Пул накопился, шанс хороший!
```

### Оптимальная ставка:

```
- Минимальная (100): малый риск, но базовый выигрыш только 10,000
- Средняя (500): баланс риск/выигрыш
- Максимальная (2000): большой риск, но базовый выигрыш 200,000!
```
