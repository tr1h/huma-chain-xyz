## ğŸ“Š SQL QUERIES FOR QUICK CHECKS

### Quick Player Count:

```sql
SELECT 
  COUNT(*) as total_users,
  COUNT(*) FILTER (WHERE created_at >= NOW() - INTERVAL '7 days') as new_7d,
  COUNT(*) FILTER (WHERE created_at >= NOW() - INTERVAL '30 days') as new_30d
FROM users;
```

### Quick Revenue Check:

```sql
SELECT 
  COUNT(*) as nfts_sold,
  SUM(CASE WHEN payment_type = 'SOL' THEN price ELSE 0 END) * 200 as revenue_usd
FROM user_nfts
WHERE minted_at >= NOW() - INTERVAL '30 days';
```

### Quick Burn/Mint:

```sql
SELECT 
  SUM(CASE WHEN type LIKE '%burn%' THEN amount ELSE 0 END) as burned,
  SUM(CASE WHEN type IN ('reward', 'mint', 'referral') THEN amount ELSE 0 END) as minted,
  ROUND(
    SUM(CASE WHEN type LIKE '%burn%' THEN amount ELSE 0 END) / 
    NULLIF(SUM(CASE WHEN type IN ('reward', 'mint', 'referral') THEN amount ELSE 0 END), 0),
    2
  ) as ratio
FROM transactions
WHERE created_at >= NOW() - INTERVAL '7 days';
```

### Quick Retention:

```sql
WITH yesterday_cohort AS (
  SELECT user_id, created_at
  FROM users
  WHERE DATE(created_at) = CURRENT_DATE - 1
)
SELECT 
  COUNT(*) as cohort_size,
  COUNT(DISTINCT t.user_id) as returned_today,
  ROUND(COUNT(DISTINCT t.user_id) * 100.0 / COUNT(*), 1) as day1_retention
FROM yesterday_cohort yc
LEFT JOIN transactions t ON yc.user_id = t.user_id 
  AND DATE(t.created_at) = CURRENT_DATE;
```
