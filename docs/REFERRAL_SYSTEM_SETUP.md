# 🔗 REFERRAL SYSTEM SETUP

## 🎯 Как работает реферальная система

### **📋 ШАГ 1: Обновить базу данных**

Выполни в Supabase SQL Editor:

```sql
-- Add referral_code to leaderboard table
ALTER TABLE leaderboard ADD COLUMN IF NOT EXISTS referral_code TEXT;

-- Create indexes
CREATE INDEX IF NOT EXISTS idx_leaderboard_referral_code ON leaderboard(referral_code);
CREATE INDEX IF NOT EXISTS idx_referrals_code ON referrals(referral_code);
```

**Или используй файл:** `SUPABASE_REFERRAL_UPDATE.sql`

---

### **🤖 ШАГ 2: Как это работает**

#### **1️⃣ Пользователь получает код:**
- Переходит в бота: https://t.me/solana_tamagotchi_v3_bot
- Пишет команду: `/ref`
- Получает свой уникальный код: `TAMA123ABC`
- Получает ссылку: `https://t.me/solana_tamagotchi_v3_bot?start=refTAMA123ABC`

#### **2️⃣ Пользователь делится:**
- Копирует ссылку
- Отправляет друзьям
- Постит в соцсетях

#### **3️⃣ Друг переходит по ссылке:**
- Открывается бот
- Автоматически отправляется `/start refTAMA123ABC`
- Бот находит реферера по коду
- Связывает нового пользователя с реферером
- **НО!** Реферал засчитается только когда новый пользователь подключит кошелек в игре!

#### **4️⃣ Что происходит в игре:**
- Новый пользователь подключает кошелек
- Игра проверяет URL параметры: `?ref=WALLET_ADDRESS&tg_id=123&tg_username=user`
- Создает запись в таблице `referrals`:
  ```sql
  INSERT INTO referrals (referrer_address, referred_address, referral_code, level, signup_reward)
  VALUES ('реферер_кошелек', 'новый_кошелек', 'TAMA123ABC', 1, 100);
  ```
- Начисляет 100 TAMA реферету
