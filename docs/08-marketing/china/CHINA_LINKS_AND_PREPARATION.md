## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### **Как работает авторизация:**

1. **Автоматическое определение:**
   - Игра проверяет доступность Telegram
   - Если Telegram недоступен → предлагает Wallet
   - Если Telegram доступен → использует Telegram

2. **Параметры URL:**
   - `?lang=zh` - устанавливает китайский язык
   - `?auth=wallet` - принудительно Wallet авторизация
   - `?auth=telegram` - принудительно Telegram авторизация

3. **Сохранение данных:**
   - Wallet авторизация → данные сохраняются в Supabase по wallet address
   - Telegram авторизация → данные сохраняются по Telegram ID
   - Данные не теряются при очистке кеша
