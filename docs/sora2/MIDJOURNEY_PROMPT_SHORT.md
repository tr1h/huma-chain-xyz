## Параметры для экспериментов:

**Размеры:**

- `--ar 21:9` - широкий формат для генплана (рекомендуется)
- `--ar 16:9` - стандартный широкий
- `--ar 2:1` - очень широкий

**Стиль:**

- `--style raw` - более реалистично
- `--style 4.0` - более художественно

**Версия:**

- `--v 6` - последняя версия (рекомендуется)
- `--v 5.2` - предыдущая версия

**Дополнительно:**

- `--chaos 20` - больше разнообразия
- `--stylize 100` - средняя стилизация
- `--quality 1` - стандартное качество
- `--quality 2` - высокое качество (больше кредитов)

## ⚠️ ВАЖНО: ВИД СВЕРХУ!

Все промты настроены на **вид строго сверху (top-down, orthographic view)** - это именно то, что нужно для генплана! Не перспектива, а план как на архитектурных чертежах.

**Ключевые слова для вида сверху:**

- `top-down view`
- `bird's eye view straight down`
- `orthographic view 90 degrees`
- `site plan layout`
- `building footprints visible from above`

## Советы:

1. **Начните с "ИДЕАЛЬНОГО ПРОМТА"** - он специально для генплана сверху
2. **Если результат хороший** - используйте расширенный промт для детализации
3. **Если что-то не так** - используйте `/remix` и уточните конкретные элементы
4. **Для вариаций** - используйте `/vary` на понравившемся результате
5. **Если получается перспектива** - добавьте `straight down 90 degrees orthographic` в промт
